# Human Aladdin

Human Aladdin 是一套「決策護欄系統」。

它不預測、不下指令、不告訴你該怎麼做，
只在錯誤變成不可逆之前，標示風險邊界。

---

## 🧠 核心理念

- 不操控人
- 不製造恐懼
- 不預言未來
- 只揭示「現在不能做什麼」
